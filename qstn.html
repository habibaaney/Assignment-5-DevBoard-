Question-1: What are the different ways to select an element in the DOM?
Different ways to select an element in the DOM are using getElementById("id") ,getElementsByClassName("class") ,getElementsByTagName("tag") ,querySelector("selector") ,querySelectorAll("selector") .
Question-2: What is the difference between innerHTML, innerText, and textContent ?
The difference between innerHTML, innerText, and textContent : innerHTML Returns or sets HTML content. innerText Returns or set text in an element. textContent Returns or set all the text content.
Question-3: What is event delegation in the DOM?
Event delegation is adding an event listener to a parent element instead of individual child elements. It improves performance and works well for dynamically added elements.
Question-4: What is event bubbling in the DOM?
Event bubbling happens when an event starts at the target element and moves up to parent elements.
Question-5: How do you create, add, and remove elements using JavaScript?
Let me give examples of creat, add and remove elements using javascript. Create: let newDiv = document.createElement("div") Add: document.body.appendChild(newDiv) Remove: document.body.removeChild(newDiv).
